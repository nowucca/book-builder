# Chapter 1: Getting Started

This is a sample chapter demonstrating the book-builder system.

## Overview

Book-builder is a reusable Pandoc + XeLaTeX build system for creating professional technical books in multiple formats:

- **PDF**: High-quality typeset books suitable for print or digital distribution
- **HTML**: Web-friendly format for online reading
- **EPUB**: E-reader compatible format

## Features

The build system includes:

- Professional LaTeX templates
- Syntax highlighting for code blocks
- Custom callout boxes for notes, warnings, and examples
- Automatic table of contents
- Bibliography support
- Image processing
